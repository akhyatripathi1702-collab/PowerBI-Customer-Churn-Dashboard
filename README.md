# Power BI Customer Churn Dashboard

## 📊 Overview
This project demonstrates data cleaning, transformation, and visualization using Power BI. It highlights customer churn, KPIs, and sales insights through interactive dashboards.

## 🔹 Features
- Data import and transformation in Power BI
- Multi‑page dashboards (churn, KPIs, city‑wise analysis)
- Exploratory visualizations for decision support

## 🛠️ Tools
- Power BI

## 🚀 How to Use
1. Clone the repository.
2. Open the `.pbix` file in Power BI Desktop.
3. Explore dashboards and visuals.

## 📈 Key Visuals
- Churn by city
- Customer segments
- KPIs and revenue trends
